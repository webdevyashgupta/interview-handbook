## 📖 Pattern-Based Revision

Master these **7 Core Patterns** from Akshay's sheet:
1. **Hashing**: O(1) membership & frequency.
2. **Two Pointers**: Moving from ends or same side.
3. **Sliding Window**: Optimal range tracking.
4. **Binary Search on Answer**: Solving optimization problems using BS.
5. **Monotonic Stack**: Finding nearest neighbors.
6. **BFS/DFS**: Traversing Trees & Graphs.
7. **DP**: Storing state to avoid redundant work.
