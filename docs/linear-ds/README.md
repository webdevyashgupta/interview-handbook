    *   Focus on basic manipulations, prefix sums, and the "Hashing" technique for frequency counting.

2.  **Phase 2: Sliding Window Mastery (Week 3)**
    *   Understand the difference between **Fixed** and **Variable** size windows.
    *   Learn the "Shrinkability" condition—when to move the left pointer.

3.  **Phase 3: The Monotonic Pattern (Week 4)**
    *   This is the "secret weapon" for Stacks. Master **Next Greater/Smaller Element**.
    *   Apply this to advanced problems like **Histogram** and **Subarray Minimums**.

4.  **Phase 4: Design & Hard Problems (Week 5)**
    *   Implement **LRU/LFU Cache** to understand how linear structures interact with HashMaps.
    *   Solve multi-technique problems like **Minimum Window Substring** and **Trapping Rainwater**.
