# NeetCode 150 — Interview Handbook

A complete, structured interview preparation guide covering all **150 NeetCode problems** organized by topic.

Each problem includes:
- 📋 **Problem Statement** with examples
- 💡 **Intuition** — the key insight that unlocks the solution
- ✅ **Optimal Solution** in Python with time/space complexity
- ⚠️ **Interview Traps** — common mistakes and follow-up questions with answers
